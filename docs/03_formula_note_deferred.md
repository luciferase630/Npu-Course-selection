# 神秘公式存档

图片中给出的公式为：

$$
(1+\alpha)\sqrt{m-n}\cdot e^{m/n}
$$

其中图片文字说明：

- $m$：选课人数；后续实验中优先按学生当时能看到的当前待选人数解释。
- $n$：课程容量。
- $\alpha$：浮动比率，范围为0.75到1.3。

## 当前处理原则

第一版不假设该公式正确，也不把它作为主模型中的理性投豆规则。

更完整的数学检查、可能推导假设和信息冲击实验设计，见 `docs/04_formula_analysis_and_information_effect.md`。

本文件只保留原始公式和来源说明，避免后续忘记公式最初的表述。

注意：学校投豆只能是整数。该公式若输出小数，只能作为连续信号存档，不能直接作为合法投豆。
